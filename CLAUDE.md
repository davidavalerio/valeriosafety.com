# valeriosafety.com

## What This Is

Redirect page. All safety consulting content has been absorbed into davidavalerio.com. This site now exists solely to redirect visitors to the professional hub.

## How It Works

`index.html` uses three redirect mechanisms for maximum compatibility:
- `<link rel="canonical">` pointing to davidavalerio.com (SEO signal)
- `<meta http-equiv="refresh">` with 0-second delay
- JavaScript `window.location.replace()` as fallback

## Files

- `index.html` — Redirect page
- `CNAME` — Points to valeriosafety.com
- `content.md` — Archived copy of original site content (reference only)
